Covid19papers is a web application aimed at helping researchers browse and comprehend a vast collection of biomedical research papers collected and distributed by Kaggle. While data scientists and AI researchers are looking for ways to satisfy questions about COVID-19 using a variety of machine-driven approaches, this resource aims to support readers with the following capabilities:

  • Perform search within the document content by keywords
  • Document search using Title, Author Name, Journal, or Paper ID
  • Generate on-demand summarization of documents
